Bem-vindo ao Gateway de Pagamentos Asaas

Microserviço para integração com a API do Asaas — rápido, seguro e flexível.

Sobre o Projeto

Esta solução permite gerenciar clientes, cobranças, pagamentos e notificações de forma centralizada, com suporte a múltiplos ambientes (sandbox e produção) e múltiplos tenants. Ideal para empresas que desejam automatizar processos financeiros e integrar facilmente com o Asaas.

Principais Recursos

Gestão de Clientes

Cadastro, consulta e atualização de clientes de forma integrada com o Asaas.

Pagamentos

Emissão e acompanhamento de cobranças e pagamentos em tempo real.

Links de Pagamento

Geração de links de pagamento personalizados para seus clientes.

Notificações

Recebimento de notificações automáticas via webhooks para eventos importantes.

Ambientes

Suporte completo aos ambientes de testes (sandbox) e produção.

Multi-tenant

Suporte a múltiplos tenants (multi-empresa) com isolamento completo.

Como Começar?

Primeiros Passos
  1. Acesse a documentação para entender os conceitos principais
  2. Configure sua conta na interface administrativa
  3. Obtenha seu token de acesso em Access Tokens
  4. Comece a integrar usando os endpoints disponíveis
