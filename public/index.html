Bem-vindo ao Payment Gateway Microservice Asaas

Microserviço para integração com a API do Asaas — rápido, seguro e flexível.
Esta solução permite gerenciar clientes, cobranças, pagamentos e notificações de forma centralizada, com suporte a múltiplos ambientes (sandbox e produção) e múltiplos tenants. Ideal para empresas que desejam automatizar processos financeiros e integrar facilmente com o Asaas.

Principais Recursos

  • Cadastro, consulta e atualização de clientes
  • Emissão e acompanhamento de cobranças e pagamentos
  • Geração de links de pagamento
  • Recebimento de notificações automáticas via webhooks
  • Ambiente de testes (sandbox) e produção
  • Documentação interativa via Swagger
  • Paginação e filtros avançados nas consultas
  • Suporte a múltiplos tenants (multi-empresa)

Como Utilizar?

Utilize o menu no topo para navegar entre as seções da documentação. Recomendamos iniciar pela Introdução para entender os conceitos principais e requisitos de integração.

Exemplo de Requisição

GET /api/customers?env=sandbox
Headers:
  access_token: SUA_CHAVE_DE_ACESSO

Tecnologias Utilizadas

Tecnologia Descrição
Node.js & NestJS Backend robusto e escalável para APIs
Swagger Documentação interativa dos endpoints
Axios Requisições HTTP performáticas
TypeScript Tipagem estática e maior segurança no código

Boas Práticas

  • Mantenha seu access_token seguro e nunca compartilhe publicamente.
  • Utilize sempre HTTPS para comunicação com a API.
  • Implemente tratamento de erros para respostas inesperadas.
  • Consulte a documentação Swagger para exemplos detalhados.
