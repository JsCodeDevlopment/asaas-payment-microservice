Interface Administrativa

Gerencie tokens de acesso e configure o gateway de pagamentos

Acesso

A interface administrativa está disponível em /admin e fornece uma interface web para gerenciar tokens de acesso e monitorar o sistema.

Funcionalidades
  • Gerenciamento de tokens de acesso
  • Visualização de logs do sistema
  • Monitoramento de status do gateway
  • Configuração de ambiente (Sandbox/Produção)
Segurança
  • A interface administrativa é protegida por autenticação
  • Use credenciais fortes para acessar o sistema
  • Mantenha os tokens de acesso seguros
  • Não compartilhe suas credenciais de acesso

Gerenciamento de Tokens

A interface administrativa permite gerenciar tokens de acesso para diferentes ambientes e clientes.

GET /admin

Acessa a interface administrativa.

Autenticação

É necessário fornecer credenciais de acesso para utilizar a interface.

Tokens de Acesso

Na interface administrativa, você pode:

  • Criar novos tokens de acesso
  • Visualizar tokens existentes
  • Ativar/desativar tokens
  • Excluir tokens
  • Gerenciar tokens por ambiente (Sandbox/Produção)
Logs do Sistema

A interface também exibe logs importantes do sistema:

  • Requisições à API
  • Erros e exceções
  • Notificações enviadas
  • Alterações em tokens

Configuração

A interface administrativa permite configurar aspectos importantes do gateway.

Ambiente

Configure o ambiente do gateway:

  • Alternar entre Sandbox e Produção
  • Configurar URLs de webhook
  • Definir timeouts e retentativas
  • Gerenciar credenciais do Asaas
Boas Práticas
  • Use o ambiente Sandbox para testes
  • Configure webhooks HTTPS
  • Monitore os logs regularmente
  • Mantenha as credenciais atualizadas
  • Faça backup das configurações

Monitoramento

A interface administrativa fornece ferramentas para monitorar o sistema.

Status do Sistema
  • Status da conexão com o Asaas
  • Status do banco de dados
  • Uso de recursos do sistema
  • Latência das requisições
Métricas
  • Número de requisições
  • Taxa de sucesso/erro
  • Tempo médio de resposta
  • Uso de tokens
